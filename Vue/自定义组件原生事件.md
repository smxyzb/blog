# 在组件中使用原生事件

## 子组件的原生事件

```
<template>
  <button @click="handleClick">点击</button>
  <input type="text" placeholder="ces" />
</template>


methods:{
  handleClick(){
    console.log('click')
  }
}
```

## 父组件使用 native 修饰符，这样就能在父组件获取到子组件的原生事件

```
<child @click.native="handleClick"  @change.native="handleChange"></child>

```
