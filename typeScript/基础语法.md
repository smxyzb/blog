# Mixed 和 any 类型
```
function add(value:mixed){
  if(typeof value === 'number'){
    value*=1
  }
  if(typeof value === 'string'){
    value.substring(0,2)
  }
}
function add(value:any){
  value*=1
  value.substring(0,2)
}
```
区别：Mixed 是强类型的，any 是弱类型的

# 数学或参数的 ?与!
## 属性或参数后面 ? ：表示可有可无
## 属性或参数后面 ! ：表示强制执行解析(告诉typescript编译器，这里一定有值)
## 变量后面 ! ：类型推断会忽略null，undefined 的情况，一定存在
