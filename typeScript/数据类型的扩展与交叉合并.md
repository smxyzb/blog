
## 扩展类型 
```
interface UserInfo {
  name:string,
  age:number,
}

interface UserWork {
  company:string
}

interface User extends UserInfo,UserWork{
  address:string
}

const user:User = {
  name:'',
  age:0,
  company:'',
  address:''
}
```

## 交叉类型 &
```
interface User1 {
  name:string,
  age:number,
}

interface User2 {
  age:number,
}

type User3 = User1 & User2 

const user:User3 = {
  name:'afff',
  age:22
}

```
